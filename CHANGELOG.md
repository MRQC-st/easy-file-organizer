# 变更日志

所有项目的显著变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [v1.0.0] - 2026-07-03

### 新增

- 批量改名：前缀文本 / 日期标记 / 编号排序 / 保留原名 四种规则
- 分类整理：按文件类型（照片/文档/视频/音频/压缩包）或按修改日期归类
- 文件操作：批量移动 / 批量复制 / 一键备份（ZIP 压缩包）
- 清理查重：MD5 哈希去重（保留最新文件）/ 清理空文件夹 / 清理冗余文件
- 数据导出：TXT 目录清单导出 / Excel（pandas）导出

### 技术

- CustomTkinter 现代 UI，左右分栏布局
- 撤销系统（最多 5 步）
- 后台线程防 UI 卡死
- 配置持久化（`~/batch_tool_config.json`）
- 拖拽文件/文件夹进入
- 按扩展名自动显示文件类型图标
